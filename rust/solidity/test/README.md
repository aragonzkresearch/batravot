# To run JS tests

1. Install Node.js
2. Install NPM modules

```bash
npm install
```

3. Run tests

```bash
npx hardhat test
```

# To run Solidity tests

1. Install Forge
2. Run tests

```bash
forge test
```

# To run benchmark

1. Install Forge
2. Run benchmark

```bash
forge snapshot
```

You will find the results in the `.gas-snapshot` file.
